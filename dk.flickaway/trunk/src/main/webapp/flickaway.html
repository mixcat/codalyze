Flickaway: a flickr viewer

Motivation: even if I like flickr, sometimes I'd like to have a simpler space for my pictures.

I wrote this little javascript program to display my pictures in a standalone html page.

    Features:
  • It's javascript-only, no server code is required.
  • Images are pulled straight away from flickr, no storage is required.
  • Supports bookmarking
  • Supports history: you can use back/forward buttons
  • All code is open source

Try the DEMO (my pictures). If you like it, you can...

Install it on your web server (with your pictures, not mine)

  • You only need to download one html file and publish it somewhere in your website
  • It will always be up-to-date: the html file is 'almost empty' and loads the required javascript and styles from the net. As I add fetaures and fix bugs, you'll always have the latest release.
  • It's skinnable, so you can style it to match your taste.
